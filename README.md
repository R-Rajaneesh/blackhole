# Blackhole

Blackhole is a DNS server running on your local machine that sucks up all your ads flowing through your network

## Installation

### Setting up
```
git clone https://github.com/r-rajaneesh/blackhole.git Blackhole

cd Blackhole

install.bat
```

## Running the server

```
npm run start
```
